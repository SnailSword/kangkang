# kangkang 
查看npm包的多级依赖(让我康康你到底依赖了啥)。

## usage
```
npm i
npm run build

node ./dist/index.js [packagename] # 查看某个发布了的npm包的依赖
node ./dist/index.js -h # 查看帮助
node ./dist/index.js cache clean # 清除缓存文件
```
